# DA SLIME SPOT

A simple Node.js YouTube downloader (MP3 / MP4 / ZIP album).

⚠️ **Important:** Downloading copyrighted content without permission may violate YouTube terms and local laws.

## Deploying to Vercel

1. Install Vercel CLI (optional):
   ```bash
   npm i -g vercel
   ```

2. Login:
   ```bash
   vercel login
   ```

3. Deploy:
   ```bash
   vercel
   ```

## Local Run

```bash
npm install
npm start
```

Then open:

```
http://localhost:3000
```
